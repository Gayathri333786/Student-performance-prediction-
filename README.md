# Student Performance Prediction System

## Overview

A Machine Learning based web application developed using Flask and Random Forest Regression to predict student academic performance.

## Features

* Student Performance Prediction
* Random Forest Machine Learning Model
* Flask Web Application
* SQLite Database Storage
* Prediction History
* Dashboard Analytics
* Responsive User Interface

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* SQLite
* HTML
* CSS
* Bootstrap

## Project Structure

* app.py
* train_model.py
* model.pkl
* templates/
* static/
* students.db

## How to Run

1. Install dependencies
2. Run train_model.py
3. Run app.py
4. Open http://127.0.0.1:5000
